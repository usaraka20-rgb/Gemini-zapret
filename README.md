# Gemini Zapret
Gemini Zapret — это удобная оболочка для управления обходом ограничений доступа к интернет-ресурсам. Скрипт автоматизирует настройку DNS и запуск сервисов обхода, делая процесс максимально простым и визуально приятным.

# Что он умеет?
Быстрое переключение: Включение и выключение режима обхода в один клик.

Умный детект: Автоматически находит твой рабочий сетевой адаптер.

Гибкие настройки: Возможность задавать собственные DNS-серверы для обхода.

Автоматизация: Удобный запуск сервисов Zapret через единый интерфейс.

Чистота: Автоматическое создание папок для логов и конфигураций (/logs, /cfg).

Защита: Проверка прав администратора для корректной работы сетевых интерфейсов.

# Для чего он нужен?
Скрипт создан для тех, кто хочет иметь простой «инструмент-переключатель» под рукой. Он избавляет от необходимости каждый раз лезть в настройки сети Windows или искать файлы запуска в папках — всё управление происходит в одном компактном и кастомизируемом терминальном окне.

# Основные преимущества  
Минимализм: Только нужные функции без лишнего мусора.

Персонализация: Легко меняй DNS под свои задачи через меню настроек.

Портативность: Вся структура папок (данные, логи, конфиги) находится рядом с исполняемым файлом.
